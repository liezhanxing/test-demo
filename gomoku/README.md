# 🎮 快乐五子棋 (Happy Gomoku)

一个适合小朋友玩的卡通风格五子棋游戏，支持双人对战！

## 🌟 功能特点

- 🎨 **卡通 UI 设计** - 蓝天白云背景，可爱棋子造型
- 🔊 **趣味音效** - 落子声、获胜音效、庆祝动画
- 👫 **双人对战** - 黑棋白棋轮流落子
- ↩️ **悔棋功能** - 下错了可以反悔
- 📱 **响应式设计** - 支持手机、平板、电脑
- ✨ **获胜特效** - 五彩纸屑庆祝动画

## 🚀 运行方式

### Web 版（推荐）
直接在浏览器打开 `index.html` 文件：
```bash
# Mac/Linux
open index.html

# Windows
start index.html

# 或双击 index.html 文件
```

### Python 原版
```bash
python3 gomoku.py
```

## 📖 游戏规则

1. ⚫ 黑棋先手，⚪ 白棋后手
2. 点击棋盘空位落子
3. 先连成五子一线（横、竖、斜均可）者获胜
4. 可以使用"悔棋"撤销上一步
5. 点击"重新开始"开始新游戏
