# 五子棋游戏 (Gomoku)

一个基于 Python 控制台的经典五子棋游戏，支持双人对战。

## 功能特点

- 15x15 标准棋盘
- 双人对战模式
- 支持横、竖、斜四个方向判断胜负
- 简单的命令行界面

## 运行方式

```bash
python3 gomoku.py
```

## 游戏规则

1. 黑棋 (X) 先手，白棋 (O) 后手
2. 输入落子位置格式：`行号 列号` (例如：`7 7`)
3. 先连成五子一线 (横、竖、斜均可) 者获胜
4. 输入 `q` 退出游戏
